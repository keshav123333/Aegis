# Aegis

Aegis is a modular AI-powered wearable safety network prototype that turns everyday wearable objects (for example bracelets, rings, or pendants) into discreet SOS devices.

## Capabilities

- Device registration for discreet wearable SOS objects
- Manual SOS triggering with real-time location payload
- Intelligent emergency detection from sensor signals
- Instant dispatch routing to:
  - Family contacts
  - Authorities
  - Nearby community responders
